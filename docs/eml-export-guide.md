# How to Export Confirmation Emails as .eml Files

TripTrace can parse .eml files directly — no OAuth setup required.
This works with Outlook, Yahoo Mail, iCloud Mail, and any other email provider.

---

## Outlook (Web — outlook.com)

1. Go to **https://outlook.com** and sign in
2. Find a flight confirmation email (search for "booking confirmation" or "itinerary")
3. Open the email
4. Click the **three-dot menu (⋯)** in the top-right of the email
5. Click **"Save as"** → saves a `.eml` file to your Downloads folder
6. Repeat for each confirmation email you want to import
7. In TripTrace, click the **"Upload .eml files"** area and select all the files

> **Tip:** You can select multiple emails in the folder list by holding Ctrl (Windows) or Cmd (Mac),
> then drag them all to your desktop at once — Outlook web doesn't support bulk download,
> but the desktop app does (see below).

---

## Outlook Desktop (Windows)

1. Open Outlook and find your flight confirmation emails
2. Select one or more emails (hold Ctrl to select multiple)
3. Drag and drop them onto your **Desktop** or a folder
   - Each email saves as a `.msg` file (TripTrace accepts both `.eml` and `.msg`)
4. In TripTrace, upload the saved files

---

## Outlook Desktop (Mac)

1. Open Outlook and find your confirmation emails
2. Select the email(s)
3. Go to **File** → **Save As** → choose **.eml** format
4. Upload to TripTrace

---

## Yahoo Mail

1. Go to **https://mail.yahoo.com** and sign in
2. Open a confirmation email
3. Click the **three-dot menu (⋯)** → **"Export"** or **"Download message"**
   - If you don't see this option, try: **More** → **View Raw Message** → copy-paste into a `.txt` file renamed to `.eml`
4. Upload to TripTrace

---

## iCloud Mail

1. Go to **https://www.icloud.com/mail** and sign in
2. Open a confirmation email
3. On the toolbar, look for the **download icon** or go to **More → Export Message**
4. The file saves as `.eml`
5. Upload to TripTrace

---

## Apple Mail (Mac)

1. Find the confirmation email in Apple Mail
2. Select it (or multiple — hold Cmd to select several)
3. Go to **File** → **Save As** → format: **Raw Message Source** (saves as `.eml`)
   - Or drag the email from Apple Mail directly to your Desktop
4. Upload to TripTrace

---

## Gmail (as an alternative to OAuth)

1. Open the email in Gmail
2. Click the **three-dot menu (⋮)** in the top-right → **"Download message"**
   - This downloads a `.eml` file
3. Upload to TripTrace

---

## Tips for finding confirmation emails quickly

Search for any of these in your inbox:

| Search term | What it finds |
|------------|---------------|
| `booking confirmation` | Most airlines and OTAs |
| `e-ticket` | Turkish Airlines, Lufthansa, Emirates |
| `itinerary` | Expedia, Kayak, Google Flights |
| `your trip` | Expedia, booking apps |
| `flight confirmation` | Delta, United, American |
| `reservation confirmed` | Various airlines |

Select all results → export all at once for the fastest import.
